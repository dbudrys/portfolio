Labas Crypto

Project:

LabasCrypto was developed using Bootstrap front-end framework to provide a responsive website that fits all screen sizes. The purpose of this project/website was to introduce people to crypto currency and promote it. LabasCrypto project was created by the request of a private customer. It includes unique features, such as: current prices of major crypto currencies, interactive scroll reveal effects, and animated home logo.
